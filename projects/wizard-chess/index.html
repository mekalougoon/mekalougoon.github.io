Brief

A first year university project, a single page desktop website built to promote a fictional extreme sports club. Tasked to use a template as a foundation, we were required to consider the target demographic and create a proffessional, engaging website that utilised at least one interactive feature using JavaScript.

We were required to change all the content of the template, as well as the order and the colours. Additionally, we were required to add sections and reorder the existing ones, as well as modifying all the visual components to best match our narrative.
